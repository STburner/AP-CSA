Secret Crossword

PURPOSE:
This project implements a class to manage crossword entries and their clues. It serves as the foundational component of a game where players guess crossword entries based on provided clues, the number of letters, and the first letter of the entry.

Version 1.0

HOW TO START THIS:
1. Open the project in BlueJ by clicking on the package file.
2. Open the `SecretCrosswordTest` class and run the `main` method to test the implementation.

AUTHORS: Me

USER INSTRUCTIONS:
1. Complete the `SecretCrossword` class as per the given specifications.
2. Use the provided test methods in `SecretCrosswordTest` to validate the implementation.
3. When all tests pass, integrate the class into the `SecretCrosswordGame` to play the game.
